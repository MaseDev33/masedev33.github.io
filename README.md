# masedev33.github.io

This repository hosts Mason's personal portfolio website. The site is built with plain HTML and CSS and includes pages for Home, About, Projects, and Contact.

The portfolio is designed for a student developer learning Python, HTML, and CSS. It keeps the layout clean, easy to read, and focused on real beginner-friendly projects.

Files in this repository include the website pages, a shared stylesheet, and site assets such as the portfolio image and favicon.
